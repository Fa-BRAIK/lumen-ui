| ------- | ------------------ |

| 10.x    | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Laravel, please send an e-mail to Farouk BRAIK via me@farouk.ca. 

All security vulnerabilities will be promptly addressed.
